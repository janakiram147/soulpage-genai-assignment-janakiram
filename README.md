# Multi-Agent System & Chatbot Assignment

## Task 1: Multi-Agent System
This task demonstrates a simple multi-agent architecture.

### Agents:
- Data Agent: Fetches company-related information
- Analyst Agent: Analyzes the data and provides insights

### Flow:
1. User enters company name
2. Data agent collects information
3. Analyst agent analyzes and returns result

---

## Task 2: Chatbot
A simple rule-based chatbot.

### Features:
- Continuous conversation
- Responds to user input
- Exit using "bye"

---

## How to Run

### Task 1:
cd task1_multi_agent  
python main.py  

### Task 2:
cd task2_chatbot  
python app.py
